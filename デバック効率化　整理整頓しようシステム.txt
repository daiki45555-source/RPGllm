# 🧹 メモリ管理ガイド（アンチグラヴィティ君へ）

## ⚠️ 問題

メモリ87%使用、Antigravityプロセス27個で2.2GB消費！
これが原因で**作業スピードが激落ち**してる！

---

## 🔍 なぜ遅くなる？
```
タブ開きっぱなし → メモリ消費 → 87%超え
    ↓
Windows「メモリ足りない...ディスクに退避するか」
    ↓
ディスクI/O爆増（HDDなら致命的）
    ↓
PC全体がモッサリ
    ↓
コード書くのも遅い、ブラウザも遅い、全部遅い！
```

---

## ✅ 今すぐやること

1. **不要なタブを全部閉じる**
2. **ブラウザを再起動**
3. **タスクマネージャーでメモリ確認（50%以下が理想）**

---

## 📋 作業中のルール

### タブ管理
```
□ localhost:3000 は1タブだけ！
□ 新しく開くな！リロード（F5）しろ！
□ 確認終わったら閉じる！
□ 同時に開くタブは5個以内！
```

### 定期メンテナンス
```
□ 1時間ごとにタブ整理
□ メモリ70%超えたら掃除タイム
□ 作業セッション終了時にブラウザ再起動
```

### やるな！
```
❌ 「後で見るかも」でタブ放置
❌ 同じページを複数タブで開く
❌ デバッグ用タブを閉じずに次の作業
```
