# 🧹 メモリ管理ガイド

> **作業前に必ずチェック！PCが遅いと全部遅くなる！**

---

## ⚠️ なぜ遅くなる？
```
タブ開きっぱなし → メモリ消費 → 87%超え
    ↓
Windows「メモリ足りない...ディスクに退避」
    ↓
PC全体がモッサリ → コード書くのも遅い！
```

---

## ✅ 作業前チェック
```
□ メモリ使用率50%以下か？（タスクマネージャーで確認）
□ 不要なタブ閉じたか？
□ localhost:3000は1タブだけか？
```

---

## 📋 作業中のルール

### タブ管理
- localhost:3000 は **1タブだけ！**
- 新しく開くな！**リロード（F5）しろ！**
- 確認終わったら閉じる！
- 同時に開くタブは **5個以内！**

### やるな！
- ❌「後で見るかも」でタブ放置
- ❌ 同じページを複数タブで開く
- ❌ デバッグ用タブを閉じずに次の作業

---

## 🎯 目標

| 項目 | 目標 |
|------|------|
| メモリ使用率 | **50%以下** |
| 開いてるタブ | **5個以下** |
