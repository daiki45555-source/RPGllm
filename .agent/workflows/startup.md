# 🏁 スタートアップ手順

// turbo-all

## 必須ステップ

1. まず `.agent/knowledge/game_lore.md` を読む
   - キャラ設定、世界観、システム、伏線を確認

2. 必要に応じて以下も確認：
   - `カストルム史.txt` - 帝国の歴史
   - `イザベラ史.txt` - マリアンヌの過去
   - `ユニークキャラと主人公.txt` - キャラ詳細
   - `伏線.txt` - 絶対守るべき伏線
   - `カルマ値の影響力　最終案.txt` - システム詳細

3. 確認後、作業内容に齟齬がないか確認してから着手

---

## ⚠️ 注意事項

- 設定と矛盾する実装をしない
- 不明点は作業前にユーザーに確認
- 新しい設定が追加されたら `game_lore.md` を更新する
