## 【作業中】ユーザーのリクエストに対応

## 【作業完了】自動セーブ（Gitコミット＆プッシュ）

作業が完了したら、以下を自動実行します：

// turbo-all

### 1. 集合知の更新（必要に応じて）
作業中に発見した新しい情報やルールがあれば `game_lore.md` に追記。

### 2. タスクリストの更新
`task.md` の完了したタスクに ✅ をつける。

### 3. Gitへ保存
```bash
git add -A
```

```bash
git commit -m "<作業内容の要約（日本語）>"
```

```bash
git push origin main
```

### 4. 完了報告
ユーザーに作業完了とGit保存を報告。
