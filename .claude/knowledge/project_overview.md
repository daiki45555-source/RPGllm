# 🎮 七つの願いが降る庭で - プロジェクト概要

## 基本情報

- **タイトル**: 七つの願いが降る庭で (In the Garden Where Seven Wishes Fall)
- **ジャンル**: LLMが紡ぐ残酷で美しいRPG
- **技術**: HTML/CSS/JavaScript + ローカルLLM (Qwen 2.5-14B)

---

## 📁 プロジェクト構造

```
RPGllm/
├── .agent/          # アンチグラヴィティ君用
├── .claude/         # Claude用（このフォルダ）
├── public/          # メインアプリ
│   ├── index.html
│   ├── style.css
│   ├── main.js
│   └── js/          # サブモジュール
├── docs/            # ドキュメント
├── tools/           # ツール類
└── 画像/            # 画像素材
```

---

## 🎯 主要システム

1. **イベントシステム** - ストーリー進行
2. **カルマシステム** - プレイヤーの選択が影響
3. **戦闘システム** - ターン制バトル
4. **インベントリ** - アイテム管理
5. **セーブ/ロード** - 進行保存

---
