# Bug: [IV3] プロローグのジャック登場シーンでUIが重なる

## 概要
プロローグイベント中、ジャック（案内役）の立ち絵とメッセージウィンドウ（dialogue-box）が重なってしまい、視認性が低下している。

## 再現手順
1. ゲームを最初から開始
2. カルマテストを終了
3. プロローグイベント（鴉の巣到着前）でジャックが登場するシーンを確認

## 期待される動作
キャラクターの立ち絵とウィンドウが適切に配置され、重ならない。

## 実際の動作
z-indexの設定またはレイアウトの幅設定により、立ち絵がウィンドウの下に隠れたり、逆に被ったりしている可能性がある。

- [ ] 原因調査
- [ ] `style.css` のレイアウト修正
- [ ] 修正確認
