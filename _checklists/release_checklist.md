# リリース前チェックリスト

新しいバージョンを公開する前に、以下の項目を確認してください。

## 動作確認
- [ ] タイトル画面からゲームを開始できるか
- [ ] キャラクター作成が正常に終了するか
- [ ] カルマテスト（評価シーン）が表示され、結果が保存されるか
- [ ] 鴉の巣（拠点）に正常に到着し、BGMが流れるか
- [ ] セーブデータを作成し、タイトル画面からロードできるか

## リソース確認
- [ ] 欠損している画像ファイル（404エラー）はないか
- [ ] BGM/SEは適切なタイミングで再生・停止されているか

## 技術確認
- [ ] ブラウザのコンソールに深刻なエラーが出ていないか
- [ ] デバッグモードが意図せず有効になったままではないか
