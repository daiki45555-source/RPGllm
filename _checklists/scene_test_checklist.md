# シーン別テスト項目

各主要シーンでの確認事項です。

## カルマテスト (Karma Test)
- [ ] 診断画面が表示される
- [ ] 選択肢がクリック可能
- [ ] 全ての質問（15問）が終了後、統計が計算される
- [ ] 背景が「カウンセリング室」になっている

## 探索 (Exploration)
- [ ] 移動ボタンが表示され、場所を切り替えられる
- [ ] 画面下の「現在地」が更新される
- [ ] バイタルゲージ（HP/MP）が表示されている

## イベント (Event Overlay)
- [ ] 背景が指定のものに切り替わる
- [ ] キャラクター立ち絵が表示される
- [ ] メッセージが正しくタイプライター表示される
- [ ] 選択肢が表示され、分岐が機能する
