# BGMカタログ

| ファイル名 | カテゴリ | 備考 |
|----|----|----|
| `title_theme.mp3` | システム | タイトル画面 |
| `ゆりかご（...）.mp3` | イベント | カルマカウンセリング診断中 |
| `鴉の巣　基本BGM.mp3` | 拠点 | 拠点（鴉の巣）のメインBGM |
| `通常バトルBGM.mp3` | 戦闘 | 一般的な戦闘シーン |
| `図書館BGM.mp3` | エリア | 図書館内 |
| `イデア　最終形態戦.mp3` | ボス | 最終決戦用 |
| `クロウ　優しさのテーマ.mp3` | キャラクター | クロウに関連するイベント |
| `ジャックの悲しみテーマ.mp3` | キャラクター | ジャックに関連するシリアスなシーン |

## 音量調整依頼（メモ）
- 「エアコン」の音はSEとして扱われていますが、少し音量が大きいため再生時に調整が必要です。
