# SE (効果音) カタログ

| ファイル名 | 用途 | 備考 |
|----|----|----|
| `カチ.mp3` | UI | カーソル移動、決定 |
| `キャンセル.mp3` | UI | 戻る、閉じる |
| `タイプライター.mp3` | イベント | テキストの表示音 |
| `剣で斬る（基本）.mp3` | 戦闘 | 近接攻撃ヒット時 |
| `ダメージSE(基本).mp3` | 戦闘 | 被ダメージ時 |
| `ピピ！.mp3` | システム | エラーや警告（または特定の通知） |
| `エアコン.mp3` | 環境音 | 特定の室内環境音（※音量注意） |

## 実装上の注意
- SEの再生は `AudioManager.playSE(path, volume)` を使用します。
